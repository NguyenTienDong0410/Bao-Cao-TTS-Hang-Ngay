## 2. Triển khai site WordPress trên LAMP Stack

### 2.1. Tổng quan về LAMP Stack

**LAMP Stack** là bộ phần mềm mã nguồn mở được sử dụng để phát triển và triển khai ứng dụng web. Tên gọi LAMP là viết tắt của bốn thành phần:

| Ký tự | Thành phần | Chức năng |
|-------|-----------|-----------|
| **L** | Linux | Hệ điều hành |
| **A** | Apache HTTP Server | Web server |
| **M** | MySQL / MariaDB | Hệ quản trị cơ sở dữ liệu |
| **P** | PHP | Ngôn ngữ lập trình phía máy chủ |

---

### 2.2. Hướng dẫn cài đặt

Trước tiên, SSH vào máy chủ với quyền root và cập nhật hệ thống:

```bash
sudo apt update -y && apt upgrade -y
```

#### Bước 1: Cài đặt Web Server Apache

Vì Apache có sẵn trong kho lưu trữ của Ubuntu 22.04, có thể cài đặt trực tiếp bằng lệnh:

```bash
sudo apt install -y apache2
```

**Một số lệnh quản lý Apache:**

```bash
systemctl start apache2      # Khởi động dịch vụ
systemctl stop apache2       # Dừng dịch vụ
systemctl reload apache2     # Tải lại cấu hình
systemctl restart apache2    # Khởi động lại dịch vụ
systemctl enable apache2     # Bật khởi động cùng hệ thống
systemctl disable apache2    # Tắt khởi động cùng hệ thống
systemctl status apache2     # Xem trạng thái dịch vụ
```

**Mở cổng 80 (HTTP) trên tường lửa:**

- Với `iptables`:
```bash
sudo iptables -I INPUT -p tcp --dport 80 -j ACCEPT
```

- Với UFW:
```bash
sudo ufw allow 'Apache'
```

**Phân quyền thư mục gốc:**

```bash
sudo chown www-data:www-data /var/www/html/ -R
```

#### Bước 2: Cài đặt MariaDB Database Server

```bash
sudo apt install -y mariadb-server mariadb-client
```

**Một số lệnh quản lý MariaDB:**

```bash
systemctl start mariadb      # Khởi động dịch vụ
systemctl stop mariadb       # Dừng dịch vụ
systemctl restart mariadb    # Khởi động lại dịch vụ
systemctl enable mariadb     # Bật khởi động cùng hệ thống
systemctl disable mariadb    # Tắt khởi động cùng hệ thống
systemctl status mariadb     # Xem trạng thái dịch vụ
```

![MariaDB chạy thành công](https://github.com/user-attachments/assets/90d23bcf-0dd7-417d-8560-5cec32795378)

**Kiểm tra phiên bản MariaDB:**

```bash
mariadb --version
```

![Phiên bản MariaDB](https://github.com/user-attachments/assets/55ef4e78-a070-439e-9839-5ec0f9dc9aa8)

**Thiết lập bảo mật cho MariaDB:**

Chạy script bảo mật để đặt mật khẩu root và loại bỏ các cấu hình mặc định không an toàn:

```bash
sudo mysql_secure_installation
```

Các tùy chọn được áp dụng trong quá trình thiết lập:
- Không chuyển sang xác thực `unix_socket`
- Đặt mật khẩu mới cho tài khoản root
- Xóa người dùng ẩn danh
- Vô hiệu hóa đăng nhập root từ xa
- Xóa cơ sở dữ liệu `test`
- Tải lại bảng phân quyền

#### Bước 3: Cài đặt PHP

Tại thời điểm thực hiện, PHP 8.1 là phiên bản ổn định mới nhất với hiệu suất vượt trội so với các phiên bản trước. Thực hiện cài đặt PHP 8.1 cùng các module cần thiết:

```bash
sudo apt install -y software-properties-common
sudo add-apt-repository ppa:ondrej/php
sudo apt install -y php8.1 libapache2-mod-php8.1
sudo apt install -y php-net-ldap2 php-net-ldap3 php-imagick \
  php8.1-common php8.1-gd php8.1-imap php8.1-mysql \
  php8.1-curl php8.1-zip php8.1-xml php8.1-mbstring \
  php8.1-bz2 php8.1-intl php8.1-gmp php8.1-redis
```

**Kiểm tra phiên bản PHP:**

```bash
php -v
```

**Kiểm tra tích hợp PHP với Apache:**

Tạo file `info.php` tại thư mục gốc của Apache:

```bash
sudo nano /var/www/html/info.php
```

Truy cập trình duyệt theo đường dẫn `http://<IP_Server>/info.php` để xác nhận PHP hoạt động.

#### Bước 4: Cài đặt phpMyAdmin

phpMyAdmin là công cụ quản lý cơ sở dữ liệu qua giao diện đồ họa, giúp thao tác với database trực quan hơn.

**Cài đặt:**

```bash
sudo apt install -y phpmyadmin
```

Trong quá trình cài đặt, chọn **Apache2** làm web server và chọn **Yes** để cấu hình database tự động qua `dbconfig-common`.

![Chọn Apache cho phpMyAdmin](https://github.com/user-attachments/assets/becbe230-e602-4d2b-806a-977160ea5557)

![Cấu hình dbconfig-common](https://github.com/user-attachments/assets/ec5a082f-ec49-4a6e-8b31-1509b1ebbe25)

Đặt mật khẩu mới cho user `phpmyadmin` và xác nhận.

![Đặt mật khẩu phpMyAdmin](https://github.com/user-attachments/assets/0e55400e-7c69-47b0-9847-bdb8e8b528ed)

**Kiểm tra phân quyền user phpmyadmin:**

```bash
sudo mysql -u root
SHOW GRANTS FOR phpmyadmin@localhost;
EXIT;
```

**Cấu hình Apache để truy cập phpMyAdmin:**

```bash
sudo ln -s /etc/phpmyadmin/apache.conf /etc/apache2/conf-available/phpmyadmin.conf
sudo a2enconf phpmyadmin.conf
sudo systemctl reload apache2
```

![Cấu hình Apache cho phpMyAdmin](https://github.com/user-attachments/assets/a3ebcfaa-c69e-44aa-9ce0-ae54a2b55c75)

**Kiểm tra:**

Truy cập `http://<IP_Server>/phpmyadmin` để xác nhận giao diện đăng nhập hiển thị thành công.

![Giao diện phpMyAdmin](https://github.com/user-attachments/assets/809187c3-c65f-4562-acae-7428bf101fae)

### 2.3. Thêm website WordPress vào LAMP Stack

#### Bước 1: Tạo thư mục chứa dữ liệu website

```bash
sudo mkdir -p /var/www/lamp.wp.com/html
sudo cp -a /tmp/wordpress/. /var/www/lamp.wp.com/html/
sudo chown www-data:www-data /var/www/lamp.wp.com/html
sudo chmod -R 755 /var/www
```

#### Bước 2: Tạo Virtual Host cho website

Sao chép file cấu hình mặc định:

```bash
sudo cp /etc/apache2/sites-available/000-default.conf \
        /etc/apache2/sites-available/lamp.wp.com.conf
```

Chỉnh sửa file cấu hình:

```bash
sudo nano /etc/apache2/sites-available/lamp.wp.com.conf
```

Thêm các dòng cấu hình:

```apache
ServerName lamp.wp.com
ServerAlias www.wp.com
DocumentRoot /var/www/lamp.wp.com/html
```

![Cấu hình Virtual Host](https://github.com/user-attachments/assets/c43346b5-12fe-47ec-b4f5-ee25006c1ed8)

**Kích hoạt Virtual Host và khởi động lại Apache:**

```bash
sudo a2ensite lamp.wp.com.conf
sudo a2dissite 000-default.conf
sudo apache2ctl configtest
sudo systemctl restart apache2
```

![Kết quả kiểm tra cấu hình](https://github.com/user-attachments/assets/cab888f5-4da2-4ba8-8e87-143d1c9e51f8)

#### Bước 3: Cài đặt SSL cho website

**Cài đặt Certbot:**

```bash
sudo apt install -y certbot python3-certbot-apache
```

**Cho phép traffic HTTPS qua tường lửa:**

```bash
sudo ufw allow 'Apache Full'
```

**Cấp chứng chỉ SSL:**

```bash
sudo certbot --apache -d lamp.wp.com
```

**Kết quả:** Website truy cập thành công qua HTTPS.
