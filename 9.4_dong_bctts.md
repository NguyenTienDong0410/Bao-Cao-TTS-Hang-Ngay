# BÁO CÁO THỰC TẬP NGÀY 9/4: Triển khai ứng dụng mail server Zimbra

---

## 1. Zimbra là gì?

Zimbra là phần mềm xây dựng Email Server được cung cấp và cài đặt dưới 2 phiên bản: mã nguồn mở (Open Source Edition) và bản có phí (Network Edition). Zimbra Collaboration Suite (Zimbra) là giải pháp hỗ trợ doanh nghiệp triển khai môi trường làm việc nhóm, quản lý và chia sẻ công việc một cách hệ thống.

---

## 2. Các tính năng nổi bật của Zimbra Mail

- **Thư điện tử:** Hệ thống thư điện tử hoàn chỉnh gồm Mail server (SMTP, POP3, IMAP, antivirus, antispam, openLDAP, backup, ...) với đầy đủ các tính năng như auto-reply, auto-forward, mail filter, ... và Mail client (Zimbra Desktop và Zimbra Web Client).
- **Lịch công tác (Calendar):** Lịch cá nhân và lịch nhóm, tự động gửi mail mời họp, ...
- **Sổ địa chỉ (Contacts):** Sổ cá nhân và sổ chung của nhóm.
- **Danh mục công việc (Task):** Của cá nhân và nhóm.
- **Tài liệu (Documents):** Tài liệu dưới dạng wiki của cá nhân hoặc soạn tập thể.
- **Cặp hồ sơ (Briefcase):** Lưu file dùng riêng hoặc chung.
- **Chat:** Chat nội bộ trong mạng LAN hoặc trên Internet.

Tất cả các mục trên đều có phần chạy trên máy chủ (nằm trong Zimbra Server), lưu trên máy chủ để có thể dùng chung và truy cập từ bất kỳ đâu có Internet. Các mục đó đều có khả năng share (kể cả các thư mục email: Inbox, Sent) cho người khác dùng chung.

Zimbra có hai phần mềm client:

- **Zimbra Desktop** (tương tự Outlook, KMail, ...): cài được trên Windows, Mac, Linux. Ngoài ra có thể dùng các mail client khác như Outlook, Evolution, KMail, Thunderbird, ...
- **Zimbra Web Client:** Giao diện với người dùng, hỗ trợ hai chế độ:
  - **Online:** Dùng Zimbra Web Client, mọi thông tin lưu trên máy chủ Zimbra. Có hai giao diện: dạng HTML thông thường (nhanh nhưng ít tính năng) và dạng Ajax (tương tự Yahoo Mail).
  - **Offline:** Dùng các mail client còn lại. Riêng Outlook, Apple Desktop và Evolution có thể đồng bộ email, calendar, contacts và task với máy chủ Zimbra.

Zimbra cũng hỗ trợ làm việc với các thiết bị di động như iPhone, Blackberry, ...

## 4. Cài đặt và sử dụng Zimbra

### 4.1 Cài đặt Email Server Zimbra trên Ubuntu 22.04

**Bước 1: Cài đặt các phần mềm cần thiết**

```bash
sudo apt-get update
sudo apt-get install curl netcat-openbsd sudo libidn11 unzip libaio1 pax
```

![Cài đặt phần mềm cần thiết](https://github.com/user-attachments/assets/1f6b741e-9d6a-4161-aa5d-f1013e888010)

---

**Bước 2: Tắt Postfix và AppArmor** *(tránh tranh chấp cổng SMTP - port 25)*

```bash
sudo service postfix stop
sudo service apparmor stop
sudo systemctl disable postfix
sudo systemctl disable apparmor
```

---

**Bước 3: Tải xuống và giải nén tệp cài đặt Zimbra**

```bash
cd /tmp
wget https://github.com/maldua/zimbra-foss/releases/download/zimbra-foss-build-ubuntu-22.04/10.1.16.p1/zcs-10.1.16_GA_4200001.UBUNTU22_64.20260310121616.tgz

tar -xvzf zcs-10.1.16_GA_4200001.UBUNTU22_64.20260310121616.tgz
```

![Tải và giải nén Zimbra](https://github.com/user-attachments/assets/67f070de-b409-436b-855b-49ac34be6012)

---

**Bước 4: Cài đặt Zimbra**

```bash
cd zcs-10.1.16_GA_4200001.UBUNTU22_64.20260310121616
./install.sh -s
```

### 4.3 Thiết lập chính sách mật khẩu (Password Policy)

```bash
su - zimbra

# Xem policy hiện tại của domain
zmprov gd dong.local | grep -i password

# Thiết lập policy cho domain
zmprov md dong.local \
  zimbraPasswordMinLength 8 \
  zimbraPasswordMaxLength 64 \
  zimbraPasswordMinUpperCaseChars 1 \
  zimbraPasswordMinLowerCaseChars 1 \
  zimbraPasswordMinNumericChars 1 \
  zimbraPasswordMinPunctuationChars 1 \
  zimbraPasswordMaxAge 90 \
  zimbraPasswordMinAge 1 \
  zimbraPasswordEnforceHistory 5 \
  zimbraPasswordLockoutEnabled TRUE \
  zimbraPasswordLockoutMaxFailures 5 \
  zimbraPasswordLockoutDuration 30m
```
